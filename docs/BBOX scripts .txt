This applies multimarkdown and uses sed to fix some rendering problems


Process Multimarkdown
	bBox_Bash( 1;  "echo \"" & quiz_mmd & "\" | mmd -f --process-html")

Process Multimarkdown with post-processing using sed
	bBox_Bash( 1;  "echo \"" & quiz_mmd & "\" | mmd -f --process-html | sed -E 's:<p>(</td></?t[dr]>)</p>:\1:g' ")

    by default this runs from the "/" directory
    see get XSLT 2.0 function in STCW198 database.
    
    
	bBox_Bash( 0; "cd " & $xml_path & " || exit ; java -jar /Applications/Saxon/saxon9he.jar " & $xml & " " & $xslt  )
